# jamesmillercvForecast
This repo hosts the Python file that analyses my website data.

1. I obtain my WordPress daily views and visitors data every day,
2. I the data into this spreadsheet: https://docs.google.com/spreadsheets/d/12XxYn4zgcHVMJfme0rvQRA4eyJe9y005nc8Jx2DM9f8/edit?gid=1047244059,
3. I hit a button in Google Sheets to trigger the connection to GitHub Actions,
4. The attached Python file processes all available data (01/01/2023 - current date),
5. GitHub Actions output to this sheet: https://docs.google.com/spreadsheets/d/12XxYn4zgcHVMJfme0rvQRA4eyJe9y005nc8Jx2DM9f8/edit?gid=33281076#gid=33281076,
6. I further process this data in Google Sheets.
